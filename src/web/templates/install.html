Velora installieren

In wenigen Schritten als App am Handy einrichten.
iPhone / iPad (Safari)
So wird Velora ein App-Icon auf dem Home-Bildschirm.
  1. Diese Seite in Safari öffnen — nicht in Chrome oder als App-Browser in Instagram/WhatsApp.
  2. Teilen-Symbol tippen (Rechteck mit Pfeil nach oben, unten in der Leiste).
  3. „Zum Home-Bildschirm" aus der Liste wählen.
  4. „Hinzufügen" rechts oben bestätigen. Velora landet als Icon.
  5. Velora vom Home-Bildschirm öffnen — dann läuft es als eigenständige App (ohne Browser-Leiste).
  6. In Einstellungen → Benachrichtigungen „Aktivieren" klicken, damit Push ankommt.
Falls das Teilen-Symbol fehlt: eine Seite nach unten scrollen lassen, Safari blendet die Leiste kurz aus.
Android (Chrome / Edge)
Einmal tippen — der Browser fragt automatisch.
Der Installations-Dialog erscheint, sobald Chrome bereit ist.
  1. Oben auf „Als App installieren" tippen.
  2. Alternativ: Menü (3 Punkte) → App installieren.
  3. Benachrichtigungen in Einstellungen → Benachrichtigungen aktivieren.
Desktop (Chrome, Edge, Safari)
Installiere Velora als Fenster-App.
  1. Chrome / Edge: Install-Icon in der Adressleiste (kleines Monitor-Symbol rechts neben der URL).
  2. Safari: Menü Datei → Zum Dock hinzufügen (ab macOS 14).
  3. Velora öffnet in einem eigenen Fenster, Push-Notifications funktionieren wie in der Handy-App.
